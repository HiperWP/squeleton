# Padding Reference

Padrão: `{breakpoint-}p-{valor}{unidade}-{direção}`

## Direções

| Sufixo | Aplicação |
|--------|-----------|
| `-all` | Todos os lados |
| `-t` | Top |
| `-b` | Bottom |
| `-l` | Left |
| `-r` | Right |
| `-tb` | Top + Bottom |
| `-lr` | Left + Right |
